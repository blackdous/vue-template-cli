## Usage

```bash
$ npm install -g vuetemplates-cli
$ vuetemplates init template-spa my-project-name
$ cd my-project-name
$ npm install
$ npm run dev
```

## Documentation

```bash
$ vuetemplates --help
$ vuetemplates init
```
